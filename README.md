# 🤖 LexBot - Telegram бот для мониторинга законодательства Узбекистана

Автоматический мониторинг Национальной базы данных законодательства **Lex.uz** с мгновенными уведомлениями в Telegram.

## 📋 Возможности

- 📡 **Автоматический мониторинг** Lex.uz каждые 30 минут
- 🔔 **Мгновенные уведомления** о новых законах, указах и постановлениях
- 📊 **Статистика** законодательства по категориям
- 🔍 **Поиск** документов по ключевым словам
- 📁 **Категории**: Налоги, Экономика, Трудовое право, IT, Гражданское право
- 💾 **SQLite база данных** для хранения документов и подписчиков

## 🚀 Установка и запуск

### 1. Клонирование и настройка окружения

```bash
cd lexbot
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

```bash
cp .env.example .env
# Отредактируйте .env файл:
nano .env
```

Заполните:
- `BOT_TOKEN` - токен от @BotFather
- `ADMIN_ID` - ваш Telegram ID от @userinfobot

### 3. Запуск бота

```bash
python lexbot.py
```

## 📱 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу, подписаться на уведомления |
| `/subscribe` | Подписаться на уведомления |
| `/unsubscribe` | Отключить уведомления |
| `/stats` | Статистика законодательства |
| `/search` | Поиск документов |
| `/categories` | Категории законов |
| `/help` | Справка |

## 📝 Структура базы данных

**documents** - хранение законодательных документов:
- id, title, doc_type, doc_number
- date_published, category, description
- url, status, created_at

**subscribers** - подписчики бота:
- user_id, username, first_name, last_name
- categories, notifications_enabled

**check_logs** - логи проверок:
- check_time, new_documents, status

## 🔄 Процесс работы

1. **Планировщик** (APScheduler) запускает проверку каждые 30 минут
2. **Парсер** получает данные с Lex.uz
3. **Сравнение** с существующими документами в БД
4. **Сохранение** новых документов
5. **Рассылка** уведомлений всем подписчикам
6. **Логирование** результатов проверки

## 🛠️ Технологии

- **aiogram 3.x** - асинхронный фреймворк для Telegram ботов
- **aiosqlite** - асинхронная работа с SQLite
- **aiohttp** - HTTP клиент для парсинга
- **APScheduler** - планировщик задач
- **python-dotenv** - управление конфигурацией

## ⚠️ Важно

- Lex.uz не имеет официального API - используется веб-скрапинг
- Соблюдайте `robots.txt` и не перегружайте сервер
- Для промышленного использования рассмотрите официальный запрос API
